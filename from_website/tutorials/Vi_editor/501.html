The Vi Editor

Robert Evans

26 th October, 2003

Copyright ©2003 R.Evans.
Email: Robert.Evans@cs.cardiff.ac.uk

Abstract:

Vi the original UNIX screen editor is available on all UNIX and Linux systems in Cardiff School of Computer Science. It is a powerful editor with a huge number of commands and options. However, it is sometimes considered hard to use.

This Note describes a few of the simple commands needed so that you can use vi to create files. It also gives a list of vi and ex (line mode) commands.
