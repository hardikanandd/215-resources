Uploading Files

The following is a set of instructions on how to upload files using the digital drop box.

  1. Go to the secure digital drop box site at https://www.cse.sc.edu.

  2. Accept the security certificate warning and log in using the same username and password that you use in lab.

  3. Once you are in the main page, click on "Drop Box - Student". You will see all drop boxes for previous assignments or any currently open assignments.

  4. Click on "Upload" for the specific assignment box you want to submit documents to.

  5. You will get a screen that has three boxes with a "Browse" button located next to each. Pay attention to the notes on that page pertaining to file naming conventions and uploading limits.

  6. Press the "Browse" button. You will get one of two windows: "Choose File" or "Browse". You should be in a folder which has the path similar to this "/acct/s2/(your username)/". This is your Home Folder which should contain your documents. The layout of the browse window might be somewhat different depending on your web browser. For example in Internet Explorer you may see a directory tree. In that case make sure you click on the folder which has your username as its name.

  7. Click on the file to be uploaded. Make sure you see the complete file path in the "File Name" box at the bottom of the window. Press "OK".

  8. Now you should see the path to the file you uploaded in the appropriate box on the upload page. DO NOT alter the file path manually. This will result in uploading an empty file.

  9. Press "Upload Files" followed by "Done".

  10. Now you should see the files you under the "Drop Box Contents" section. If you have uploaded the wrong file or an old version of it, press "Delete" to remove it from your drop box.

Transferring Files from a Floppy to Your StarOffice Work Folder

  1. Logon on the front computer in either of the Sumwalt computer labs.

  2. Insert the floppy disk in the floppy drive.

  3. Click on the arrow above the folders image on the toolbar at the bottom of your desktop.

  4. Select Open floppy and wait until the contents of your floppy are displayed in a window.

  5. Repeat the same process this time selecting StarOffice work folder. Now you have both the floppy and the work folder windows open.

  6. From the floppy window drag and drop the icon for the desired file into the work folder. You may do the reverse if you wish to transfer a file from the work folder to a floppy.

  7. You must eject the floppy properly otherwise the next floppy inserted in the drive will be corrupted and could not be used. To eject properly, in the floppy window click on File and the Eject.

  8. Close both windows.

Converting File Types

  1. To do this, you simply open the file to be converted.

  2. Click Save As. When the Save window opens, leave the file name as is and select the desired file type you want to convert to.

  3. Example: If you want to convert a MS Word document (.doc extension) to StarWriter document (.sxw extenstion), open the Word document, click Save As, and choose StarWriter 6.0/7 for the file type. This will result in retaining your original Word document and creating a StarWriter version of it additionally.

Terminating a Frozen Application (e.g. StarOffice, Netscape)

  1. From your desktop, click on the arrow above the icon that resembles two colored columns. Click on "This Host".

  2. Type the following at the command prompt: ps -Uusername. Enter your own username in place of "username" in this command.

  3. You will see the list of processes you are currently running on your machine. Observe the PID of the application you want to terminate.

  4. Type at the following at the command prompt: kill -9 PID. The PID in this command is the PID number you observed for the application you want to kill.

  5. Now you may start the application.

  6. Alternatively, from the toolbar on your main desktop click on the arrow above the fourth icon from the right. Click on Find Process. In the window opened you will see a box called Filter. Type in your username in the Filter box. This will display the process you are running. Find the frozen process by looking under the Name column (e.g. netscape if your Netscape window is frozen or soffice for Staroffice). Click on it, then right click, and select kill. This should terminate the process.

How to Display Web Pages Using Your USC Web Space

The following is a set of instructions/ requirements that must be fulfilled in order to view web pages on the USC network.

  1. Move all your HTML pages (.html extension) in addition to all images you used to your public_html folder. To do this open your home folder. You will see a folder icon named public_html. Drag and drop the desired files onto the public_html folder icon. Do not transfer the files that have .html extension followed by additional ~ characters.

  2. Open the public_html folder (double click on it) from the home folder window. If you have not already saved your main page as index.html, you will see that there is a default index.html file there, delete it (right-click, choose put in trash). Now click on the icon of your main page once and click on the name part to get a cursor in the name. Change the file name to index.html.

  3. index.html is the first (or main) page that others will see when they type in your URL. In order for them to see your other pages, you MUST provide links on the first page to the other pages. In other words, you should have hyper links to each of the other pages.

  4. Next, you should check your URL to make sure that everything can be viewed OK. Your USC network URL is cl.cse.sc.edu/~username. Make sure you put your own username there.

  5. If any of the images that used to show up before is not showing now, you must check three things:
    • Go to the public_html folder and make sure that the image file is there.

    • If the image is there, right-click on it, select "Properties". You should have the Read attribute checked for Owner, Group, and Other.

    • One other possible problem if the above two are not an issue is .tiff files. If your Tiff image is not showing, open the image and do "Save As", and save it with a .jpg extension. Remember that this means you should modify the filename in your code accordingly.

    • Go back and check Reload (Refresh) your page and see if the images show.
