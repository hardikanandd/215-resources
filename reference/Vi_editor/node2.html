Vi

The application vi (vee-eye) is the standard editor on UNIX operating systems. It is available on all UNIX systems and can potentially run on any computer with a monitor or vdu screen. On Linux systems, vi is a mode of the even more powerful editor, vim. Vim is context sensitive. If it recognises the type of file it is editing it will highlight parts of the text in colour to show you features of the content. For example, if the file is a C program, it can highlight reserved words, strings, blocks, etc.

In general, the description here of vi also applies to vim in vi mode.

The editor vi can be used in terminal emulation windows on UNIX or Linux workstations, or on terminal emulation windows connected remotely to UNIX or Linux from other systems such as Windows PCs.

It is a terminal-screen based editor, ie it shows part of file that's being edited in the terminal emulation window. You edit the file by editing the characters in the terminal. When you are happy with the edits, you write them away to the file. Up to that point, the file on disk is untouched.
