Moving About in a File

The arrow keys on the keyboard
,
,
and
, can be used to move the cursor up, left, right or down when vi is in editing mode. If you try to move above or below the lines in the terminal window, the lines in the screen terminal are rolled down or up to reposition the window on the file.

You can also locate the cursor by context. The command:

/string

will move the cursor to the next occurrence of string. Similarly,

?string
will move the cursor to the previous occurrence of string. There are also ``go to line n'' commands. The command
:n
moves the cursor to line n in the file. In particular,
:1
moves the cursor to the start of the file and
:$
moves it to the last line in the file.
